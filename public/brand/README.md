# Kit de marca — turnogym

Todos los archivos del logo de la plataforma, en un solo lugar.

## Colores

| Uso | Color |
|---|---|
| Cian eléctrico (principal) | `#22D3EE` |
| Azul (degradé) | `#3B82F6` |
| Cian profundo (para fondo claro) | `#0EA5C4` |
| Verde volt (acento opcional) | `#A3E635` |
| Fondo oscuro | `#0B0F16` |
| Texto claro | `#E6EDF3` |

## Archivos

| Archivo | Qué es | Cuándo usarlo |
|---|---|---|
| `turnogym-icon-cian.svg/.png` | Ícono cuadrado, degradé cian→azul con la T-mancuerna oscura | App, favicon, redes (versión principal) |
| `turnogym-icon-oscuro.svg/.png` | Ícono cuadrado oscuro con la T-mancuerna blanca | Cuando el fondo es claro o querés algo sobrio |
| `turnogym-glifo.svg` | Solo el símbolo T-mancuerna, sin fondo | Marcas de agua, sellos; cambiá el `fill` al color que quieras |
| `turnogym-wordmark-oscuro.svg/.png` | Logotipo "turnogym" para **fondo oscuro** (turno blanco + gym cian) | Header oscuro, web, redes |
| `turnogym-wordmark-claro.svg/.png` | Logotipo "turnogym" para **fondo claro** (turno oscuro + gym cian) | Documentos, facturas, fondos blancos |

## Notas

- Los **SVG** son la fuente (vectoriales, se agrandan sin perder calidad y se
  pueden editar). Los **PNG** son para pegar rápido donde no acepten SVG.
- El monograma "TG" **no se usa**: el ícono oficial es la T-mancuerna.
- La tipografía del wordmark es una sans geométrica redondeada (tipo Poppins).
  Los PNG del wordmark pueden verse con una fuente aproximada; para el render
  exacto usá el SVG o el componente `components/BrandMark.tsx` de la app.
- Tagline: **"Es tu turno."**
